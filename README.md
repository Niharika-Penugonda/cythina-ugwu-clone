🌐 Cythina Ugwu Portfolio Clone

This project is a frontend recreation inspired by the award-winning portfolio website of Cythina Ugwu.
The goal of this project was to practice building modern UI layouts, smooth interactions, and creative website structures using core frontend technologies.

As a beginner frontend developer, I built this project to improve my understanding of how creative developer portfolios are designed and implemented.

🚀 Project Overview

This project focuses on recreating the visual layout and interactive feel of a modern creative portfolio website. It helped me learn how to structure websites that emphasize clean design, smooth scrolling, and engaging user experience.

🛠️ Technologies Used

HTML5 – Structure of the website

CSS3 – Styling and layout

JavaScript – Interactivity and dynamic behavior

🎯 What I Learned

While building this project, I improved my skills in:

Structuring modern website layouts

Creating interactive UI components

Working with animations and transitions

Improving problem-solving in frontend development

Understanding how creative portfolios are built

📂 Project Purpose

The purpose of this project is learning and practice.
It helped me gain hands-on experience with frontend development and build confidence in creating visually appealing websites.
